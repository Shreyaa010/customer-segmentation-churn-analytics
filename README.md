# Customer Segmentation & Churn Analytics in European Banking

## Project Overview

This project analyzes customer churn patterns in a European banking dataset. The objective is to identify high-risk customer segments, understand churn behavior across demographics and geography, and provide business insights for customer retention strategies.

## Objectives

- Measure overall churn rate
- Analyze churn by geography
- Compare churn across age groups
- Evaluate tenure and engagement patterns
- Identify high-value customer churners
- Perform customer segmentation using K-Means Clustering

## Dataset Features

- Credit Score
- Geography
- Gender
- Age
- Tenure
- Balance
- Number of Products
- Credit Card Status
- Active Member Status
- Estimated Salary
- Exited (Churn Indicator)

## KPIs

- Overall Churn Rate
- Segment Churn Rate
- High-Value Churn Ratio
- Geographic Risk Index
- Engagement Drop Indicator
